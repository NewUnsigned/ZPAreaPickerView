# ZPAreaPickerView
用于区域选择的控件

###### 导入文件
<pre><code>
#import "ZPAreaSearchPickView.h"
</code></pre>

###### 使用方法

<pre><code>
    ZPAreaSearchPickView *pick = [ZPAreaSearchPickView cityPickViewWithFrame:CGRectMake(0, 100, self.view.width, 88) title:@"区域选择"];
    [self.view addSubview:pick];
</code></pre>
